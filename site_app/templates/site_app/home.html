Invitation to tango

I looked at my son. He's well-known front-end developer and I like very much what he makes. And I asked myself how all of these websites work and can I understand it? Can I make something as wonderful as my son makes? It was a warm rainy autumn day (2018) when I decided to learn Python and Django from zero.

I would say it's my tango with Django. Now I'm for a while on this road and I still keep going... I try to implement simple Python & Django functions and result of it you can see at this site. In the blog you can see short history how is it going right now...


Functionality & technologies

Site has various sections witch are different Django applications:

  • Blog
  • LocalLibrary
  • Study exercises
  • Search

All posts in the Blog are from server's database. It can be SQLight or PostgreSQL/MySQL database. It depends from server where site is running. Everyone can read and add posts.

At LocalLibrary you can see function and generic class based views, access restriction, different permission by different group of users as result of implementing decorators and mixins. If you want test behavior of users' use Authentication form at very bottom of page.

Study Exercises show some techniques of templates in Django, content of GET Request and others.
