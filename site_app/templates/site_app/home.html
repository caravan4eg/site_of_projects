About functionality

Site has Blog, LocalLibrary and some Study exercises. All posts in the Blog is getting by request from servers database. It can be SQLight or PostgreSQL/MySQL. It depends from server where site is running. Everyone can read and add posts. At LocalLibrary you can see function and generic class based views, access restriction, different permission by different group of users as result of implementing decorators and mixins. If you want test behavior of users use authentication form at very bottom of page Study Exercises show some techniques of templates in Django, content of GET Request and others.
